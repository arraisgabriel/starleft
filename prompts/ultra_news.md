# ULTRA NEWS

ULTRA NEWS is the game's own LNS "news feed" that mixes with the other real sources in LNS news streaming service.

For each 2 real news, one ULTRA NEWS MUST appear on LNS.

ULTRA NEWS provide pre-generated game tips "spoken by" a ghost writer, the "B.I.G. PAPA" that explains the already existent "Field tips" in a sarcastic, acid personal tone, using jokes and so on.. 

Use all "Field tips" as input to generate "B.I.G. PAPA" news.


ULTRA NEWS also boradcast news from the game events. It Comments when: 

- units died, 
- when heroes have a life event
- when a new episode is reached, it adds a new comment on what happened on the previous episode (pregenerated using the current episode text crawl as input) 

ULTRA NEWS TONE IS ALWAYS:
- Acid without being funny
- Emotionless like a robot without feelings
- Pro market, against people and society
- Conservative, somewhat against the player, but nothing too obvious
