# Music system

Revamp the current music system (only available on main menu) to work now in all game, main menu, episode maps and hub map.

Every map has it's own set of musics. 
Every music has a main and a loop file. The loop file MUST be played right after the main theme starts. Then after some minutes the main theme must start again and so on, in a infinite loop.

The new system MUST have in-game controls to toggle the music and adjust it's volume and voices volume as well.

The new system MUST have events trigger to start different tracks. This system will be used for instance to change the music when a battle starts in the middle of a mission
