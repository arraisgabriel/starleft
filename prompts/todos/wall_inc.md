# Wall inc

Wall inc is the new name of this game (by analubussular)

## Motivation

Starleft if a goofy name that does not entirely match this game dark tone.


## The new name - WALL INC
Wall inc is the name of the company that the player controls in the game (the red faction).

Wall also summarizes this game core mechanics, unit carrying / evolving (leveling-up, implants..) / dying, throughout the game. The wall in the gasme is constantly referenced as the list of dead soldiers. 

## Pink Floyd the Wall
Wall is also a reference to pink floyd's the wall album, in particular the another brick in the wall music.
